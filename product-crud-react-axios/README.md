# Product CRUD React (Vite)

This is a simple React + Vite project implementing a Product CRUD with:

- React Router (routing)
- Login page (simple hard-coded validation: admin/admin)
- Add / Edit / Delete / List
- Functional components + hooks

## Setup

1. Install dependencies:
```bash
npm install
```

2. Start dev server:
```bash
npm run dev
```

3. Open the URL shown by Vite (e.g., http://localhost:5173)

## Notes

- This project uses in-memory state (no backend). To persist data, integrate localStorage or a backend API.
